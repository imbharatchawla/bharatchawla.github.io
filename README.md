# Bharat Chawla Portfolio

A GitHub Pages-ready static portfolio built with HTML, CSS and JavaScript.

## Features

- Fully responsive desktop, tablet and mobile layout
- Dark/light mode with saved user preference
- Typewriter-style hero animation
- Terminal-inspired developer card
- Resume download button
- Experience, skills, projects and contact sections
- No build step required

## Files

```text
index.html
styles.css
script.js
assets/Bharat-Chawla-Resume.pdf
.nojekyll
```

## Deploy on GitHub Pages

1. Create a new GitHub repository, for example `bharat-portfolio`.
2. Upload all files from this folder to the repository root.
3. Make sure the homepage file is named `index.html`.
4. Go to repository **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select branch `main` and folder `/root`, then click **Save**.
7. Your site will publish at a URL like:
   `https://your-username.github.io/bharat-portfolio/`

## Customize

- Update email, phone or location in `index.html`.
- Add GitHub/LinkedIn links in the Contact section if required.
- Replace `assets/Bharat-Chawla-Resume.pdf` when you update your resume.
- Change colors in `styles.css` using the variables inside `:root`.
